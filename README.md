# Ranjeet Portfolio

A modern and responsive personal portfolio website showcasing my projects, skills, experience, and achievements in full-stack web development.

## 🚀 Features

* Modern and responsive UI
* About Me section
* Skills and technologies showcase
* Featured projects
* Contact section
* SEO optimized
* Fast performance with Next.js
* Mobile-friendly design
